# ⚡ Exit Rules - Quick Reference

## หุ้นจะหายจาก Portfolio เมื่อไหร่?

### 🔴 ออกทันที (CRITICAL)

| Rule | เงื่อนไข | ตัวอย่าง |
|------|---------|----------|
| ✅ **TARGET_HIT** | กำไร ≥ **+4.0%** | ซื้อ $100 → ขาย $104+ |
| ❌ **HARD_STOP** | ขาดทุน ≤ **-3.5%** | ซื้อ $100 → ขาย $96.50 |
| 📉 **TRAILING_STOP** | ตกจาก peak **-3.5%** | Peak $110 → ขาย $106.15 |

### 🟠 สัญญาณแรง (HIGH)

| Rule | เงื่อนไข |
|------|---------|
| 🔻 **Gap Down** | เปิดตลาด gap down ≥ 1.5% + ขาดทุน > 1% |
| 📉 **Breaking Down** | ตกในวัน ≥ 2% + ขาดทุน > 0.5% |

### 🟡 สัญญาณปานกลาง (MEDIUM)

| Rule | เงื่อนไข |
|------|---------|
| 📊 **Volume Collapse** | Volume < 50% avg + ขาดทุน > 1% |
| 🎢 **Failed Pump** | เคยขึ้น > 3% แล้วตกต่ำกว่าราคาซื้อ |
| 📉 **SMA20 Break** | ราคา < SMA20 ≥ 1% + ขาดทุน |

### 🟢 สัญญาณอ่อน (LOW)

| Rule | เงื่อนไข |
|------|---------|
| 😔 **Weak RSI** | RSI < 35 + ขาดทุน > 2% |
| 🔄 **Momentum Reversal** | ขึ้นแล้วกลับตก |
| ⏰ **MAX_HOLD** | ถือครบ **30 วัน** |

---

## 💡 คำถามด่วน

**Q: ขาดทุน -2% จะออกไหม?**
A: ยังไม่ออก (ต้องถึง -3.5% หรือถูก SMART rules)

**Q: กำไร +3% จะออกไหม?**
A: ยังไม่ออก (ต้องถึง +4%)

**Q: Gap down -1.6% + ขาดทุน -1.2% จะออกไหม?**
A: **ออก!** (SMART_GAP_DOWN)

**Q: ถือ 30 วัน ขาดทุน -1% จะออกไหม?**
A: **ออก!** (MAX_HOLD)

---

## 🎯 Key Points

1. **Rule แรกที่ตรง → ออกทันที**
2. **SMART exits ออกก่อนถึง -3.5%** (avg -2.98%)
3. **CRITICAL rules เช็คก่อน** (target, stop, trailing)
4. **Backtest: 40% win rate, profitable** ✅
