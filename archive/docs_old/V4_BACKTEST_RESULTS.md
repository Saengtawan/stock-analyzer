### 📈 Alt Data vs ผลตอบแทน

| Alt Data | จำนวน | Win Rate | Avg Return | Predictive? |
|----------|-------|----------|------------|-------------|
| 0/6 | 1 หุ้น | 100% | +10.5% | ไม่ชัด |
| 1/6 | 2 หุ้น | 100% | +25.9% | ไม่ชัด |
| 2/6 | 7 หุ้น | 100% | +23.1% | ไม่ชัด |

**💡 สังเกต:** ไม่มีความสัมพันธ์ชัดเจน
- Alt Data 0/6 → +10.5%
- Alt Data 1/6 → +25.9% (สูงกว่า!)
- Alt Data 2/6 → +23.1%

**ไม่มีรูปแบบที่สม่ำเสมอ** → ไม่ predictive!

### 🎯 เหตุผลที่ Alt Data ไม่สัมพันธ์

**1. Alt Data เป็น Lagging Indicator**
- ข้อมูล insider trading, analyst ratings มักล่าช้า
- ตลาดตอบสนองก่อนที่ข้อมูลจะออกมา
- Momentum เป็น leading indicator (บอกก่อน)

**2. Alt Data Coverage ไม่ครอบคลุม**
- ไม่ใช่ทุกหุ้นจะมี alt data ครบ
- EXAS มี alt data แค่ 1/6 แต่ทำได้ +45.8%!
- META มี alt data 0/6 แต่ทำได้ +10.5%!

**3. Momentum ดีกว่า**
- Momentum (RSI, MA50, Mom30d) บอกแนวโน้มจริง
- Alt data บอกเฉพาะ sentiment (อาจผิด)
- Momentum correlation: +0.542 vs Alt data: +0.141

### ✅ สรุป: ทำไมถึงไม่สัมพันธ์?

**Alt Data ≠ การทำกำไร** เพราะ:
1. ล่าช้า (lagging)
2. ไม่ครอบคลุม (limited coverage)
3. Sentiment ≠ Price Action
4. Momentum เป็นตัวชี้วัดที่ดีกว่าเยอะ

**v4.0 ทำถูกแล้ว** ที่ทำให้ alt data เป็น "bonus" ไม่ใช่ "requirement"!
