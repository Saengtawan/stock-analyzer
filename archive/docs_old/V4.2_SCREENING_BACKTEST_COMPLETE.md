## 🔬 v4.2 ANTI-EXTENDED CRITERIA

```
✅ Momentum:     ANY positive, but <38% (reject extended)
✅ RSI:          45-72 (reject both weak and overextended)
✅ Volume:       0.8-1.8x (reject extremes)
✅ MA20 > MA50:  Uptrend required
✅ 5d momentum:  > -8% (not in freefall)
```

### Why v4.2 Works:
1. **Rejects Extended Moves**: Momentum <38%, RSI <72 prevents buying exhausted rallies
2. **Requires Momentum**: Stocks must have positive momentum (but not excessive)
3. **Healthy Volume**: 0.8-1.8x ensures liquidity without panic/euphoria
4. **Uptrend Filter**: MA20 > MA50 ensures structural strength
5. **No Freefall**: 5d momentum > -8% avoids catching falling knives

## 🛡️ NEW TRAILING STOP (Breakeven Protection v4.2)

### How It Works:
```python
if highest_price > entry_price:  # Activate on ANY profit!
    profit_pct = ((highest_price - entry_price) / entry_price) * 100

    if profit_pct >= 5.0:
        trailing_pct = -3.0      # Tight trailing for big gains
    elif profit_pct >= 2.0:
        trailing_pct = -4.0      # Moderate trailing
    else:
        # Breakeven protection: max(-1.0, -(profit_pct - 0.5))
        trailing_pct = max(-1.0, -(profit_pct - 0.5))
```

### Impact:
- **Triggered in**: 94.2% of trades (114/121)
- **Protected Profits**: 3 of bottom 10 losers were actually profitable at peak!
