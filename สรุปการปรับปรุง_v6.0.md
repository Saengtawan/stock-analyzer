## ✅ การแก้ไขที่ทำ

### 1. ปรับ "Sweet Spot" ของ Gap ให้แคบลง

**ก่อนแก้:**
- Gap 2.0-3.5% → +15 คะแนน (ช่วงกว้างเกินไป)
- Gap 3.5-4.5% → +8 คะแนน (ยังให้รางวัลกับ gap ที่เสี่ยง)

**หลังแก้:**
- Gap 2.0-3.0% → +12 คะแนน ✅ (sweet spot ที่แคบลง)
- Gap 3.0-3.5% → +5 คะแนน (ลดรางวัล)
- Gap 3.5-4.5% → 0 คะแนน (เป็นกลาง แทนที่จะให้รางวัล)
- Gap 4.5-7.0% → **-10 คะแนน** (ลงโทษ - ช่วงใหม่)
- Gap >7.0% → **-20 คะแนน** (เพิ่มโทษจาก -15)

### 2. แก้ปัญหา "Overbought" (สำคัญที่สุด!)

**ก่อนแก้:**
```
Consistency ≥80% → +15 คะแนน (ปัญหา: ให้รางวัลกับหุ้นที่ร้อนเกินไป!)
```

**หลังแก้:**
```
Consistency ≥95% → -5 คะแนน (เตือน: OVERBOUGHT!)
Consistency 80-95% → +8 คะแนน (ดี แต่ลดจาก +15)
Consistency 60-80% → +10 คะแนน (SWEET SPOT ใหม่!)
```
